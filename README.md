# La Maison des Pyjamas

Site officiel et back-office de La Maison des Pyjamas.

## Statut

Initialisation du dépôt pour connecter GitHub à Supabase.

## Stack cible

- Next.js 16 App Router
- TypeScript strict
- Tailwind CSS v4
- shadcn/ui
- Supabase / PostgreSQL
- Supabase Auth
- Supabase Storage
- Zustand pour le panier
- TanStack Query
- React Hook Form + Zod
